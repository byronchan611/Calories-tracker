# PokeCoach

## Phase 1 Features ✅

This application provides a simple and flexible way to track bodyweight and daily nutrition.

### Bodyweight Tracking
Log your bodyweight over time.
Visualise progress with a line graph displaying changes across different days.

### Meal Tracking
Record daily meals by entering macronutrient values (calories, carbohydrates, fats, protein).
Create and reuse preset meals built from custom ingredients.

### Nutritional Calculations
Automatically calculates total daily intake:
- Calories
- Carbohydrates
- Fats
- Protein
- 
### Custom Ingredient Database
Add and store nutritional information for ingredients.
Customise values based on specific brands or store sources for improved accuracy.

### Macronutrient Distribution
View how your daily calories are split across carbs, fats, and protein.
Displayed in an easy-to-read pie chart.

### History Tracking
All daily entries are saved under the History tab.
Review past nutrition and weight data at any time.
